Задание.

Написать структуру даных студентов курса ЕПАМ.
Атрибуты студента:
- фамилия (стринг)
- имя (стринг)
- профессия (кодируется одной заглавной буквой)
- вес (инт, точность 2 знака после запятой)

Функционал для реализации:
- добавление студента
- удаление студента
- отображение списка всех студентов (только фамилии и в порядке добавления в список)
- смена местами студентов в списке
- отображение полной информации о студенте путем ввода его порядкового номера

* при написании кода использовать try/catch

** доп.функционал:
- сохранение списка студентов в файл
